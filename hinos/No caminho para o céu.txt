NO CAMINHO PARA O CÉU

Quem está andando aí, com tal peso sobre si?
Deixe o fardo, pecador, no caminho para o céu!

Coro:
Eu quero caminhar, o meu fardo vou deixar;
Jesus comigo sempre andará.
Louvemos glória, aleluia!
Seguindo eu vou cantar. O sol sempre a brilhar,
Venha andar comigo, no caminho para o Céu.

Jovens de mãos dadas vão, com os anjos cantarão.
Velhos andam sem cansar, no caminho para o Céu.

Não há mais tristeza lá, choro não existirá;
Lágrimas já não tem mais, no caminho para o Céu.
